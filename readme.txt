== Description ==

This add-on allows you to add subscribers to your MailChimp lists, with forms made with [FormCraft](http://formcraft-wp.com/). You can read the tutorial [here](http://formcraft-wp.com/help/how-to-configure-mailchimp-add-on/)

Please note that this add-on works with [FormCraft Premium](http://formcraft-wp.com/), and not [FormCraft Basic](https://wordpress.org/plugins/formcraft-form-builder/)
